## OrchRec: Intent-Driven Orchestration Recommendation

This repository contains the implementation of **OrchRec**, a multi-agent framework for intent-driven recommendation of orchestration solutions across the Cloud–Edge–IoT continuum.

---

## Evaluation Results

The complete evaluation results, including LLM-as-a-judge and human evaluation data, are available in the Evaluation results folder. 
The results include:
- LLM-as-a-judge evaluation (multiple configurations and scenarios)
- Human evaluation (expert-based assessment)
- Scenario-level and criterion-level analysis
- Qualitative feedback

---

## Implementation

This repository contains the backend prototype implementing the proposed approach, including:
- Intent interpretation
- Knowledge graph querying
- Reasoning and coverage verification
- Recommendation generation

---

## Reproducibility

The provided code and evaluation results allow full reproducibility of the experiments presented in the paper.
